## Installation

Clone the project

```bash
  https://github.com/nil0412/SocialMediaDashBoard_MERN.git
```

Go to the project directory

```bash
  cd SocialMediaDashBoard_MERN
```

Install dependencies

```bash
  npm install
```
## Run Locally

Start the server

```bash
  npm start
```


## Features

- Login/Register User
- Create Post
- Like & RemoveLike on Posts
- Add Comments to Posts
- Edit own profile
- Seach other Users
- View Others Profile
- Add & Remove Friends
